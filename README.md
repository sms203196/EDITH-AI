# Edith AI 🤖

Edith AI is a Python-based virtual assistant that helps with automation tasks, web requests, and basic user commands.

## 🚀 Features
- Automation using PyAutoGUI  
- Web requests handling  
- Wikipedia search support  
- Basic user command execution  

## 🛠️ Technologies Used
- Python  
- PyAutoGUI  
- Requests  
- Wikipedia module
